# Product Overview

**Quyosh Paneli Ma'lumotlari** (Solar Panel Info App) is a Flutter-based mobile and web application for displaying solar panel information in Uzbek language.

## Core Features

**User-facing:**
- Browse solar panel types with specifications (power, efficiency, warranty, price)
- View pricing packages and comparisons
- Access contact information
- Offline mode with local caching
- Automatic update notifications

**Admin panel:**
- Secure login system
- Manage panels (add, edit, delete)
- Manage pricing packages
- Update contact information
- View usage statistics

## Target Platforms

- Android (primary)
- Web
- Windows
- iOS (configured but not primary focus)

## Language

All UI text, comments, and documentation are in Uzbek (Uzbek Latin script). Variable names and code structure follow English conventions.
